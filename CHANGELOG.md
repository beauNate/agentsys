### Added
- Initial repository setup
- Zero-configuration platform detection system
- Five core slash commands: `/ship`, `/next-task`, `/deslop-around`, `/project-review`, `/pr-merge`

- Support for multiple CI/CD platforms (GitHub Actions, GitLab CI, CircleCI, Jenkins)
- Support for multiple deployment platforms (Railway, Vercel, Netlify, Fly.io)
- Automatic project type detection (Node.js, Python, Rust, Go)
- Context-efficient git command optimizations
- Expandable slop detection patterns
- Framework-specific review patterns
- Comprehensive documentation
